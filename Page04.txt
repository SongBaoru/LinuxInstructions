Vim是一个基于文本界面的编辑工具，是所有Linux发行版本默认的文本编辑器
Vim是Vi的增强版，在Vi的基础上增加了正则表达式的查找、多窗口的编辑等功能
Linux 系统中所有的内容都以文件的形式进行存储，当在命令行下更改文件内容时，常会用到文本编辑器
Vim有三种工作模式 ：命令、输入、编辑


5.3
1、Vim三种模式
	1)Vim命令模式
		使用 Vim 编辑文件时，默认处于命令模式
	
	2)输入模式
		在命令模式状态下输入i l a A o O等插入命令
		各指令及具体功能 ：
			i : 在当前光标位置插入文本，光标后的文本相应向右移动
			l : 在光标所在的行首插入文本
			o : 在光标所在行的下一行插入新行文本
			O : 在光标所在行的上一行插入新行文本
			a : 在当前光标所在位置之后插入随后输入的文本
			A : 在光标所在的行尾插入文本
			
	3)编辑模式
		对文件中的指定内容执行保存、查找或替换等操作
		使Vim切换到编辑模式的方法是在命令模式状态下输入:
		指令执行后 Vim 会自动返回命令模式。如想直接返回命令模式，按 Esc 即可
		

5.4 -- 5.6
Vim基本操作
1、打开文件
		vim aa : 打开文件aa
		打开文件的命令 ：
			vim filename : 打开或新建一个文件，并将光标置于第一行的首部
			vim + filename : 打开文件，并将光标置于最后一行的首部
			vim -r filename : 恢复上次 vim 打开时崩溃的文件
			vim -R filename :  	把指定的文件以只读方式放入 Vim 编辑器中
			vi + filename : 打开文件，并将光标置于第 n 行的首部
			vi +/pattern filename : 打幵文件，并将光标置于第一个与 pattern 匹配的位置
			vi -c command filename : 在对文件进行编辑前，先执行指定的命令
			
2、编辑文件(输入模式 + 编辑模式 + 命令模式)
	1)插入文件的命令 ：
		快捷键 ：
			a : 在当前光标所在位置之后插入随后输入的文本
			l : 在光标所在的行首插入文本
			A : 在光标所在的行尾插入文本
			o : 在光标所在行的下一行插入新行文本
			O : 在光标所在行的上一行插入新行文本
			i : 在当前光标位置插入文本，光标后的文本相应向右移动
				
		esc : 退出输入 
			
	2)查找目标文本的命令 ：
		快捷键 ：
			:/abc : 从光标所在位置向前查找字符串abc，如果想忽略大小写，则输入命令":set ic"
			:？abc : 从光标所在为主向后查找字符串abc
			:/^abc : 查找以abc为行首的行
			:/abc$ : 查找以abc为行尾的行
			:n : 向同一方向重复上次的查找命令
			:N ：向相反方向重复上次的查找指定
		
		如果在字符串中出现特殊符号，则需要加上转义字符 "\"，如要查找字符串 "10$"，则需要在命令模式中输入 "/10\$"。

	3)替换文本命令
		快捷键 ：
			r : 替换光标所在位置的字符
			R ：从光标所在位置开始替换字符，其输入内容会覆盖掉后面等长的文本内容，按“Esc”可以结束
			:s/a1/a2/g : 将当前光标所在行中的所有 a1 用 a2 替换
			:n1,n2s/a1/a2/g : 将文件中 n1 到 n2 行中所有 a1 都用 a2 替换
			:g/a1/a2/g : 将文件中所有的 a1 都用 a2 替换
		
		:g/root/wangfei/g ：将文件中所有的root替换为wangfei
		:%s/root/wangfei/g : 同上:g/root/wangfei/g，将文件中所有的root替换为wangfei

	4)删除文本
		快捷键 ：
			x ：删除光标所在位置的字符
			dd : 删除光标所在行
			ndd : 删除当前行（包括此行）后 n 行文本
			dG : 删除光标所在行一直到文件末尾的所有内容
			D : 删除光标位置到行尾的内容
			:a1,a2d : 函数从 a1 行到 a2 行的文本内容

		被删除的内容并没有真正删除，都放在了剪贴板中。将光标移动到指定位置处，按下 "p" 键，就可以将刚才删除的内容又粘贴到此处

	5)复制和粘贴文本
		yy : 将光标所在行复制到剪贴板
		yw : 将光标位置的单词复制到剪贴板
		p(小写) : 将剪贴板中的内容粘贴到光标后
		P(大写) : 将剪贴板中的内容粘贴到光标前
		y : 复制已选中的文本到剪贴板

	6)其他常用的快捷键
		:j : 两行合并成一行，下行合并到当前行
		
	7)保存退出文本
		:wq : 保存并退出Vim编辑器
		:wq! : 保存并强制退出 Vim 编辑器
		:x : 保存文本，并退出 Vim 编辑器，更通用的一个 vim 命令
		:x! : 保存文本，并强制退出 Vim 编辑器，更通用的一个 vim 命令
		:q : 不保存就退出 Vim 编辑器
		:q! : 不保存，且强制退出 Vim 编辑器
		:w : 保存但是不退出 Vim 编辑器
		:w! : 强制保存文本
		:w filename : 另存到 filename 文件
		ZZ : 直接退出 Vim 编辑器

3、光标的移动
	1)快捷方向键
		h : 光标向左移动一位
		j : 光标向下移动一行
		k : 光标向上移动一行
		l : 光标向右移动一位

	2)光标以单词为单位移动
		w或W ：光标移动至下一个单词的单词首
		b或B ：光标移动至上一个单词的单词首
		e或E ：光标移动至下一个单词的单词尾
		nw或nW : n 为数字，表示光标向右移动 n 个单词
		nb或nB : n 为数字，表示光标向左移动 n 个单词
		
	3)光标移动到行首或行尾
		0或^ : 光标移动至当前行的行首
		$ ：光标移动至当前行的行尾

	4)光标移动至指定字符
		fx ：光标移动至从光标处到当前行尾中下一个 x 字符处
		
	5)光标移动到指定行
		gg : 光标移动到文件开头
		G : 光标移动至文件尾行首
		nG : 光标移动到第 n 行，n 为数字
		:n ：将光标快速定义到指定行的行首

	6)光标移动到匹配括号处
		% : 先将光标移到一个括号处，然后使用%命令，随之光标定位到另一个括号
		
4、撤销和回复撤销
	u : 撤销最近一次对文本做的修改操作
	Ctrl+R : 恢复最近一次所做的撤销操作
	U : 第一次会撤销对光标所在行做过的全部操作，第二次使用该命令会恢复撤销
